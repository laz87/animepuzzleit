# **App Name**: AnimeSlide

## Core Features:

- Classic Puzzle: Implement classic sliding puzzle mechanics on a 3x3 grid.
- Image Display: Display the full anime image for reference.
- Smart Shuffle: Generate a solvable puzzle configuration for each new game.
- Move Counter: Track the number of moves made by the player.
- Timer: Measure and display the time taken to solve the puzzle.
- Best Score: Save and display user best scores.
- Sound Management: Implement sound effects with an option to toggle sound.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) for a calming, anime-inspired aesthetic.
- Background color: Very light grayish-blue (#F0F8FF) for a clean and airy feel.
- Accent color: Pale Goldenrod (#EEE8AA) to highlight interactive elements.
- Font: 'Poppins', a geometric sans-serif (headline and body), for a modern and fashionable appearance.
- Use simple, line-based icons with rounded corners for controls.
- Implement smooth tile sliding animations using Framer Motion with a slight bounce effect.
- Mobile-first responsive design with a square grid layout, scaling appropriately on different screen sizes.
